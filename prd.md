# 目标
我要做一个中国房屋市场趋势分析的项目，主要分析中国房屋市场的趋势，包括房屋价格、房屋销售量、房屋购买量等，提供相关的趋势分析报告、图表、数据等功能，同时要求项目可以通过网页访问（包括PC端和移动端）,先做上海市的分析，同时保留其他城市数据分析的拓展能力

# 数据来源
项目的主要数据来源包括：
1. 网上房地产(https://www.fangdi.com.cn/)
2. 上海银行间同业拆放利率(https://www.shibor.sh.cn/shibor/)
3. 上海公积金官网(https://www.shzfgjj.cn/)

# 功能
项目的主要功能包括：
1. 分析中国房屋市场的趋势，包括房屋价格、房屋销售量、房屋购买量等。
2. 提供中国房屋市场的趋势分析报告，包括房屋价格趋势、房屋销售量趋势、房屋购买量趋势等。
3. 提供中国房屋市场的趋势分析图表，包括房屋价格趋势图、房屋销售量趋势图、房屋购买量趋势图等。
4. 提供中国房屋市场的趋势分析数据，包括房屋价格数据、房屋销售量数据、房屋购买量数据等。

# 技术栈
1. 分析是否有必要本地存储数据，如果需要则要求前后端分离，后端语言使用Go语言、数据库使用MySQL，其余技术栈全部由AI根据项目需求选择最优方案，如果不需要存储数据则全部在前端实现
2. 用户不做登陆注册，直接访问项目即可，但要防止用户直接访问后端接口，导致数据泄露，同时限制同一IP地址访问频率，防止DDoS攻击
3. 项目最终要求可以部署到vercel上

# 页面功能
1. 首页：展示中国房屋市场的趋势分析报告、图表、数据等功能入口，用户可以通过首页快速访问到相关功能，并展示各子页最核心内容，并且可以筛选城市以及一手房、二手房等类型；
2. UI要求现代化，具有精致感和高级感，桌面和手机端都要好看，同时主页面和各子页面风格一致
## 各个子页功能
1. 成交数据：首先要有一个日历，展示选中城市以及房子类型每一天的成交数据，并单独给一个汇总栏用于展示该月份的成交数据量，成交数据内容包括：成交房价、成交销售量、成交面积、成交总金额等
2. 趋势分析报告：展示中国房屋市场的趋势分析报告，包括房屋价格趋势、房屋销售量趋势、房屋购买量趋势等，用户可以通过报告快速了解中国房屋市场的趋势，趋势既展示城市整体趋势情况，也要展示城市下不同区域的趋势情况
3. 楼市利率：展示中国房屋市场的利率数据，包括贷款利率（涵盖公积金贷款、商业贷款等）、首付比例，用户可以通过利率数据了解中国房屋市场的利率趋势
